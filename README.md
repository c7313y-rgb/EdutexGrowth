# Edutex Growth Vision Map v4

相関図を中核に、Edutex全事業を俯瞰できるGitHub Pages対応Webアプリです。

## 公開手順
1. ZIPを解凍
2. `index.html`, `style.css`, `app.js`, `.nojekyll`, `assets/` をGitHubリポジトリ直下へアップロード
3. Settings > Pages > Branch: main / root で公開

## 主な機能
- 添付相関図の組み込み
- 成長ストーリーの訴求
- 事業カード一覧
- 検索・領域フィルタ
- 詳細モーダル
- JSON/CSV出力
